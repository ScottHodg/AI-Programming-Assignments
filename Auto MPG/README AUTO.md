# **Vehicle Fuel Efficiency (MPG) Regressor**

This project focuses on building a regression model to predict the miles per gallon (MPG) of vehicles based on their technical specifications.

## **Project Overview**

As required by Assignment 1 of the AI Programming course, this project implements:

* **Exploratory Data Analysis (EDA)**: Programmatic analysis of vehicle data.

* **Deep Learning**: Training a keras.models.Sequential DNN regressor to predict MPG.

* **Inference**: Using the trained model to predict efficiency for a specific set of new vehicle data.

## **Prediction Data**

The model is tested against specific vehicle parameters, including:

* Cylinders  
* Displacement  
* Horsepower  
* Weight  
* Acceleration  
* Model Year

Found in Auto Scott Hodgins.ipynb, the project contains:

* **Model Training**: A sequential neural network architecture optimized for regression.

# **AI Programming Assignments**

This repository contains machine learning and deep learning projects completed for the AI Programming (MBAI 5310G) course at Ontario Tech University. It covers a range of techniques, including regression, classification, and natural language generation using Deep Neural Networks (DNN) and Recurrent Neural Networks (RNN).

## **Repository Overview**

### **1\. Abalone Sex Classification**

* **Objective**: Predict the sex (Male, Female, or Infant) of an abalone based on physical measurements.

* **Key Tasks**:  
  * Programmatic data analysis of the UCI Abalone dataset.

  * Implementation of a baseline "poor" DNN versus an optimized Sequential DNN.

  * Data preprocessing involving categorical mapping and feature scaling.

* **Files**: Abalone Scott.ipynb

### **2\. Vehicle MPG Regression**

* **Objective**: Predict vehicle fuel efficiency (Miles Per Gallon) using technical specifications.

* **Key Tasks**:  
  * Exploratory Data Analysis (EDA) on vehicle attribute data.

  * Training a keras.models.Sequential regressor.

  * Inference testing on new vehicle samples (including 2025 model year predictions).

* **Files**: Auto Scott Hodgins.ipynb

### **3\. English Text Generator**

* **Objective**: Build a character-level text generator trained on classic literature from Project Gutenberg.

* **Key Tasks**:  
  * Preprocessing a corpus combined from three major literary works.

  * Training an RNN-based architecture (LSTM/GRU/Simple RNN) from scratch without the use of pretrained transformers.

  * Generating a 100-word sample of synthetic text.

* **Files**: Books.ipynb
